grep - (Global Regular Expresssion) The command is used to search the word in the file.

awk - it is used to extract specific column from the text file i.e. awk '{ print $1 }' file.txt

sed - it is a stream editor used to searching, replacing, modiying and deleting text in a file i.e. s=substitue, g=global, i=insert, d=delete, p=print

systemctl - it is used to check the services status

journalctl - it is used to check the logs.

ps - to check the process

top/htop - to check the cpu usage

free - to check the memory

uname -r - to check the current version of the kernel

ls - list the current files and folders

pwd - print working directory \n

cd - change directory

cat - to view the content of file

touch - to create the empty file

top/htop - to view the running processes

head - to check the first 10 lines of file

tail = to check the last 10 line of files

cp - to copy the file

mv - move or rename the folder

chmod - to change the permission of file

chown - to change the owner of file

sudo - superuser do (to login as root)

df -h - disk free usage

du -sh - to check the directory size in the file system

ping - to check the internet connectivity

wget - to download the files from internet

ss -tulnp or netstat -tulnp - to check the port

curl - client url (to interact with api's and website directly)

ifconfig and ip a - to check the ip address

nslookup or dig - to obtain information about domain names , ip address and other dns records (name server lookup)

whoami - current user
